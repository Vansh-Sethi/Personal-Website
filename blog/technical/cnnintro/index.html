A Beginner’s Guide Convolutional Neural Networks

2020-02-12

Everything you need to know to get started with CNNs

The range that computer vision applies to in modern society is broad. We use computer vision to diagnose major diseases. We use computer vision to unlock our phones.We even use computer vision as wacky snapchat filters. It was the creation of the Convolutional Neural Network that revolutionized the computer vision field and gave the ability for computers to “see.”

How Computers see Images

You and I see images by using light that hits our retinas and then processing the light in our brains to produce what we see. Computers don’t have this ability to read qualitative data such as light and interpret it in an abstract way to create an image. However, computers are really good with numbers and we give computers the ability to see by transferring it into this medium.

Particularly, we break up images into three main color channels, red, green and blue. This is where the term RGB comes from. Each color channel is made up of a grid of pixels that have a value attached to them. The value represents how intense a certain pixel is, for example if there was a pixel in the blue channel, the value would tell you how blue the pixel is. Each value is measured out of 255, a seemingly arbitrary number but is 255 specifically because it can easily be represented in binary. The red, green and blue colour channels are stacked on top of one of each other and a combination of a pixels’ red, green and blue channels can give us a variety of colours. We essentially can quantify images now, which makes it much easier for computers to understand and see images.

“Building Facial Detection Software With JavaScript”, 2018

What are CNNs and Why are they Special?

CNNs is a type of neural network. A neural network is a concept derived from machine learning that mimics the functions of our brain to give the ability to computers to learn. CNNs are used in machine learning and artificial intelligence problems and is one of the more well known neural networks.

Traditional object detection and image classification hinges on the idea that the thing being analyzed is in the same place for all images. Or, that it maintains a constant part of the image. This is a niche method of going about object detection and image classification. CNNs solve this problem by being able to recognize objects regardless of their position. CNNs introduce the ability to find objects in spatially different images.

How CNNs Work

Filters

A filter is a defined N x M (N & M represents the size of the grid) grid of numbers that is multiplied with the original image multiple times. It is slid across the image by a predetermined *stride *value and is multiplied with the current numbers. Every time the filters grid values are multiplied by the image grid values, the output for each value is summed together and is put into a new grid called the convolved matrix. This process is generalizing characteristics of an image by using the values of the filters. The values of the filters are determined in the training phase of the model, which is where the ‘machine learning’ aspect of the CNN comes to play. The filters are the heart of convolutional neural networks.

“Convolutional Filters, a Popular Choice”, 2018

Pooling

The pooling layer is used to generalize the features extracted from the convolutional layer and represent it in a compressed way. There are many different ways to pool the convolved matrix but the most used pooling method is max pooling. Max Pooling takes the largest number in a predefined section of the convolve matrix, essentially taking the most significant part of that section of the image.

Padding

One of the issues that arises with CNNs is the fact that filters can’t operate on edges, and there will be some parts of the filter not actually on a pixel value. Padding solves this by adding a ring of extra pixels around the original image, so the filters can process the full image. The value of the extra pixels that are added are 0. The reason for this is because 0 essentially ignores that value and represents no significance to the filter, essentially acting as a placeholder.

Dropout

To make sure that the parameters of the CNN will not memorize the dataset and actually generalize data, dropout layers are used. Dropout layers turn off filters periodically to make sure that the weight of the parameters is shared equally among all the filters. If all the parameters play a decently sized role in the model, then the model will have an easier time generalizing.

Training

With all the basic parts of the CNN layed out, the real fun finally begins. Training is what allows CNNs to actually work by giving them the ability to find the best parameters to get the best accuracy possible. The training phase consists of two steps, the loss function and the optimizer.

The loss function calculates how bad the model did. It does this by taking the predicted output and comparing it to the actual output (label). It uses some functions (there are many) and outputs a total loss. The most common loss for CNN classification is categorical cross-entropy. The higher the loss, the worse the model did. Knowing this, if we can minimize this loss and get lower numbers, then our model will do better. To do this, we use optimizers.

The optimizer minimizes the loss function by computing the gradient. The gradient is like instantaneous rate of change in calculus, and essentially tells you which direction to go in to minimize the loss. It then proceeds to go in the direction where the loss is lower. This process of loss function and optimizer is repeated over multiple epochs until the developer is pleased with the result.

Applications of CNNs

Image Classification

CNNs are really good at classifying images into different categories. The extracted features from the convolution layers allows this to happen. We have been able to classify a multitude of things to cancerous cells and brain tumors to simple dogs and cats separator.

Image Segmentation / Object Detection

Image Segmentation is image classification and figuring out where a certain object is in within the image. From CNNs, you can extract the feature maps of an image. Feature maps are the main features of an image, and dense layers can figure out the coordinates on the screen that pertain to the object. It can then also classify it. Things like snapchat filters run on this algorithm.

“Object Detection With CNN, RCNN And Fast RCNN”, 2019

GANs

Using the feature maps produced by CNNs, we can create new images from them. To do this, we use a GAN or a general adversarial network which takes in noise and produces an image out of it. It learns how to do this by using a CNN to figure out how the image should look and common patterns between all images. (very simplified explanation).
