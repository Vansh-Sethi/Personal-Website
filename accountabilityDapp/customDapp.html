Project Description

Rethinking Task Management with the Blockchain

Decentalized applications or dapps are protocol based applications that work indefinitly without a third party. A dapp was created for people to hold themselves accountable for their own tasks.

Solidity

Using the Ethereum blockchain's protocol language, the main functionality was built with decentralization in mind.

React.js

The user-interface was built with the React.js framework developed by Facebook. A multitude of components were created.

Truffle

The dapp was tested and deployed with the use of the tools in the Truffle suite. In particular, Ganache was used to set up a fully custom blockchain to test the application.

How it Works

The Process

How the decentalized application was built.

1

Smart Contracts

The principal functionality / protocol was built with smart contracts. This allows for the user to make transactions governed by a completely trustless source. The smart contracts were made in Solidity and can allow a user to create a task with a certain stake and allows for the user to donate money to a charity if they don't finish said task.

2

Testing the Application

Testing software that runs on a blockchain is hard to do. That's why a personal localized blockchain was created with the help of Ganache. This allows for testing capabilities and to ensure that the application works indefinitly. The smart contracts were deployed with the Truffle suite onto the personal blockchain.

3

User Interface

To ensure that the user can easily interact with the application, a variety of tools were used. The Web3.js framework was used to communicate with the blockchain and allow for transactions to occur. It sends JSON-RPC requests to the blockchain to recieve data or carry out certain procedures. The React.js framework was used to build a component based website, to allow for tasks to be displayed.
